## 作者

同济大学管理科学与工程系：羊山、王子路、马畅、孙岩

## 说明

页面设计：参考了马蜂窝、Airbnb的设计

Logo设计：猫的图标为原创，圆形图标非本人设计，来源Dribbble

后端框架：腾讯Wafer2

图片来源：马蜂窝

前端框架：Vue.js

网站：非商用，仅为软件开发作业

## 目录说明

`\Web` 前端目录

`\Server` 后端目录

## 本地测试说明

本地需要安装 **Node.js**，以下命令行在 `\Web` 文件夹下运行

### 安装依赖

```
npm install
```

### 本地测试

```
npm run dev
```
